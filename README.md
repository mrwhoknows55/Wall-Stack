# Wall-Stack

## About the App
A wallpaper app which uses wallheaven's api to fetch cool wallpapers and helps to set them as homescreen wallpaper.   
This app is created in #30DaysOfKotlin challenge. 

Demo [Video](docs/wallstack.mp4)
